# 更新日志

这里记录项目中值得关注的用户可感知变更。

## [0.1.2] - 2026-04-22

### 新增

- 翻译阶段新增 `google-free`、`dashscope`、`openai-compatible` 多渠道支持，默认翻译渠道改为 `google-free`。
- TTS 阶段新增 `vllm-omni` 渠道支持，可接入自部署的兼容服务。

### 改进

- `podtran init` 改为按所选翻译/TTS 渠道引导配置，只在实际需要时要求填写对应的 API key 或 `base_url`。
- 重构 TTS 配置结构，区分 `preset` 与 `clone` 模式配置，并对旧版 TTS 配置提供自动重建和备份。

### 修复

- 优化翻译与 TTS 阶段的失败和中断处理，兼容的部分结果现在可通过 `podtran resume` 继续执行。
- 收紧 `synthesize` 和 `compose` 的前置检查，避免在翻译或合成结果不完整时继续产出不一致的结果。

## [0.1.1] - 2026-04-16

### 新增

- 新增 `podtran version` 子命令，用于查看当前安装版本。

### 改进

- TTS 阶段支持多线程处理，提升合成阶段效率。
- 优化 `podtran init` 的默认翻译配置，改为使用默认非思考模型以提升执行效率。

## [0.1.0] - 2026-04-15

### 新增

- 发布播客分阶段翻译 CLI 的初始可用版本。
